# Configuration Services Catalog

SQL services used by the configuration skill for system configuration queries.

## Primary Services

| Schema | Service Name | SQL Object Type | Purpose |
|--------|-------------|-----------------|---------|
| QSYS2 | SYSTEM_VALUE_INFO | VIEW | System values with current and default settings |
| QSYS2 | HARDWARE_RESOURCE_INFO | VIEW / TABLE FUNCTION | Hardware resource inventory |
| SYSTOOLS | CONFIGURATION_STATUS | VIEW | Controller, device, and line status |
| QSYS2 | ENVIRONMENT_VARIABLE_INFO | VIEW | System and job environment variables |
| QSYS2 | JVM_INFO | VIEW / TABLE FUNCTION | Active Java Virtual Machine information |
