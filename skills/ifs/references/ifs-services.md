# IFS Services Catalog

SQL services used by the ifs skill for Integrated File System operations.

## Primary Services

| Schema | Service Name | SQL Object Type | Purpose |
|--------|-------------|-----------------|---------|
| QSYS2 | IFS_OBJECT_STATISTICS | TABLE FUNCTION | List/search IFS objects with metadata |
| QSYS2 | IFS_OBJECT_PRIVILEGES | TABLE FUNCTION | Authority settings for IFS objects |
| QSYS2 | IFS_READ | TABLE FUNCTION | Read text file contents |
| QSYS2 | IFS_OBJECT_LOCK_INFO | TABLE FUNCTION | Lock information for IFS objects |
| QSYS2 | SERVER_SHARE_INFO | VIEW | NetServer file and print shares |
| QSYS2 | COMPARE_IFS | TABLE FUNCTION | Compare IFS objects between systems |
| QSYS2 | IFS_OBJECT_REFERENCES_INFO | TABLE FUNCTION | Reference counts for IFS objects |
| QSYS2 | IFS_JOB_INFO | TABLE FUNCTION | Jobs using IFS objects |
| SYSTOOLS | IFS_ACCESS | SCALAR FUNCTION | Check IFS access permissions |
