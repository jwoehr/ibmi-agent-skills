# Journal Services Catalog

SQL services used by the journal skill for journal management and analysis.

## Primary Services

| Schema | Service Name | SQL Object Type | Purpose |
|--------|-------------|-----------------|---------|
| QSYS2 | JOURNAL_INFO | VIEW | Journal properties, receiver counts, object counts |
| QSYS2 | JOURNAL_RECEIVER_INFO | VIEW | Receiver timestamps, sizes, entry counts |
| QSYS2 | JOURNALED_OBJECTS | VIEW | Objects being journaled |
| QSYS2 | DISPLAY_JOURNAL | TABLE FUNCTION | Read journal entries |
| QSYS2 | REMOTE_JOURNAL_INFO | VIEW | Remote journal configuration and lag |
| QSYS2 | SMAPP_ACCESS_PATHS | VIEW | System-managed access path protection |
| QSYS2 | AUDIT_JOURNAL_DATA_MART_INFO | VIEW | Audit data mart status |
