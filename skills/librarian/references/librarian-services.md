# Librarian Services Catalog

SQL services used by the librarian skill for library and object management.

## Primary Services

| Schema | Service Name | SQL Object Type | Purpose |
|--------|-------------|-----------------|---------|
| QSYS2 | LIBRARY_LIST_INFO | VIEW | Current job library list |
| QSYS2 | OBJECT_STATISTICS | TABLE FUNCTION | Object details (size, owner, type) |
| QSYS2 | OBJECT_PRIVILEGES | VIEW | Object authority details |
| QSYS2 | AUTHORIZATION_LIST_INFO | VIEW | Authorization lists and secured objects |
| QSYS2 | AUTHORIZATION_LIST_USER_INFO | VIEW | User authorities on authorization lists |
