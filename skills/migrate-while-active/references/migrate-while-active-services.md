# Migrate While Active Services Catalog

SQL services used by the migrate-while-active skill for monitoring migration environments.

## Primary Services

| Schema | Service Name | SQL Object Type | Purpose |
|--------|-------------|-----------------|---------|
| QSYS2 | MIGRATION_MANAGER_INFO | VIEW | Overall migration state and phase |
| QSYS2 | LIBRARY_MIGRATION_LIST | VIEW | Library objects being tracked |
| QSYS2 | IFS_MIGRATION_LIST | VIEW | IFS objects being tracked |
| QSYS2 | MIGRATED_LIBRARY | VIEW | Successfully migrated library objects |
| QSYS2 | MIGRATED_IFS | VIEW | Successfully migrated IFS objects |
| SYSIBMADM | DATA_COMPRESSION_FACTOR | TABLE | Compression ratios |
| QSYS2 | ESTIMATE_FINAL_SYNCHRONIZATION_TIME | TABLE FUNCTION | Final sync time estimate |
| QSYS2 | MIGRATION_NETWORK_BANDWIDTH | TABLE FUNCTION | Network bandwidth measurement |
| QSYS2 | CHANGE_MIGRATION_MANAGER | PROCEDURE | Modify migration settings (not used) |

## Key Column References

- ENTRY_TYPE, SOURCE_NODE_STATE, COPY_NODE_STATE
- MIGRATION_STAGE, MIGRATION_PHASE, PENDING_MIGRATION_PHASE
- MIGRATION_PHASE_DETAIL, MIGRATION_DETAIL_INFO_TEXT
- MIGRATION_INTERVAL_COUNT, QUALIFIED_JOB_NAME
- SOURCE_NODE, COPY_NODE, PRIMARY_NODE

### LIBRARY_MIGRATION_LIST

- INITIAL_TRACKING_TIME, OBJECT_SIZE, MIGRATION_START_TIME

### IFS_MIGRATION_LIST
- PATH_NAME, OBJECT_TYPE, TRACKING_STATE
